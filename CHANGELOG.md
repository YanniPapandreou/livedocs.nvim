## [0.1.0] - 2025-11-01

### Added
- Initial release of livedocs.nvim
- Live LSP hover documentation in persistent split window
- Automatic documentation updates on cursor movement with 200ms debounce
- Flexible window positioning (above, below, left, right)
- `ToggleDocs` command with optional position argument
- Configuration options for UI, window, and buffer settings
- Markdown syntax highlighting for documentation
- Health check support (`:checkhealth livedocs`)
- Integration with Neovim's built-in LSP
